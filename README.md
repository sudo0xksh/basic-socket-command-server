Basic Socket Command Server is a Python-based TCP server that
accepts a connection, sends commands, and prints responses.

It is designed to demonstrate how interactive socket communication works.

---

## What This Tool Teaches

- TCP socket setup
- Client-server communication
- Blocking input/output flow
- Why authentication and validation matter

---

## Usage

python server.py <host> <port>

Example  
python server.py 127.0.0.1 4444

---

## Important Notes

- This tool has no authentication
- Do not expose to the internet
- Use only in controlled lab environments

---

## Final Thought

Understanding sockets is essential.
Securing them is even more important.
